Install / load

For local development, build the extension and load the generated `dist/` folder in Chrome’s Extensions page.

⌨

Keyboard shortcut

The overlay uses the `Alt+H` shortcut to toggle the writing layer while you are on YouTube.

⇄

Sync issues

If app pairing is not working, make sure the extension and app are signed in and the current device is paired.

Frequently asked questions

How do I open the notes dashboard?

Open the extension and click the dashboard / notes button, or visit the main Slatify site if you expose that link publicly.

Why does the extension need YouTube access?

Slatify is built around YouTube study sessions, so it needs access on YouTube pages to show the overlay and capture notes in context.
